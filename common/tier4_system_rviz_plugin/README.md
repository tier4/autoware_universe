## Tier4_system_rviz_plugin

## 目的

このプラグインはAutowareからハザード情報を表示し、初期のローカライズとルート設定に起因する緊急事態時に通知を出力します。

## 入力

| 名前                              | 型                                             | 説明                                                      |
| --------------------------------- | ------------------------------------------------ | ----------------------------------------------------------- |
| `/system/emergency/hazard_status` | `autoware_system_msgs::msg::HazardStatusStamped` | Autowareからの緊急情報に関するトピック |
