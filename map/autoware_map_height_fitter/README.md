## autoware_map_height_fitter

このライブラリは、与えられた点と、ポイントクラウドマップの地面にフィットさせます。
パラメータ`map_loader_name`で指定されたノードの`enable_partial_load`パラメータで、マップのローディングが切り替えられます。
このライブラリを使用したノードは、マルチスレッドの実行を使用する必要があります。
